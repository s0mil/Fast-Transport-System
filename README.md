# FAST NU Transport Management System (Front-End)

A responsive and modern web interface for managing university transport routes, built with pure HTML, CSS, and JavaScript.

## Features
- **Dark Theme UI** with neon accents and glassmorphism.
- **Live Digital Clock** in the header.
- **Route Cards** with expandable stop lists (Nadeem & Jadoon routes).
- **Search Functionality** (filter by driver, route, or stop name).
- **Category Filters** (All, Nadeem, Jadoon).
- **Dynamic Stats** (Total routes, drivers, stop counts).
- **Persistent Booking System** (Uses LocalStorage to save bookings).

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (ES6+, DOM Manipulation, LocalStorage)

## How to Run
Simply open the `index.html` file in any modern web browser.

## Project Structure
- Main layout
- All styling
- Logic, rendering, search, and booking
- Mock data for routes
